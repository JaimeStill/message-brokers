# [Getting Started](https://www.rabbitmq.com/tutorials)

These tutorials cover the basics of creating messaging applications using RabbitMQ.

1. [Hello World](./01-helloworld/) - The simplest thing that does *something*.
2. [Work Queues](./02-work-queues/) - Distibuting tasks among workers (competing consumers pattern).
3. [Publish / Subscribe](./03-publish-subscribe/) - Sending messages to many consumers at once.
4. [Routing](./04-routing/) - Receiving messages selectively.
5. [Topics](./05-topics/) - Receiving messages based on a pattern (topics).
6. [Remote Procedure Call (RPC)](./06-remote-procedure-call/) - Request / Reply pattern.
7. [Publisher Confirms](./07-publisher-confirms/) - Reliable publishing with publisher confirms.
